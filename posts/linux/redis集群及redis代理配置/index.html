Include try.conf

Include cluster.conf

   Password i4ZHIJNDYvndeZOh
   MasterReadPriority 60
   StaticSlaveReadPriority 50
   DynamicSlaveReadPriority 50
   RefreshInterval 1
   ServerTimeout 1
